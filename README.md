 # Nyoni-movie Player

A sleek, modern video player with playlist functionality.

## Features
- Play/Pause controls
- Volume control & mute
- Video playlist
- Keyboard shortcuts
- Responsive design
- Time display

## Keyboard Shortcuts
- **Space/K** - Play/Pause
- **M** - Mute/Unmute
- **Left Arrow** - Rewind 10s
- **Right Arrow** - Forward 10s

## How to Use
1. Clone/download this folder
2. Add your own video files
3. Update the `videos` array in `script.js`
4. Open `index.html` in a browser

## Video Requirements
- MP4 format recommended
- Add video files to project folder
- Update src paths in JavaScript
